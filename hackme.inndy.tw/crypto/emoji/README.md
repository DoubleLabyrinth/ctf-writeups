You can see that `encrypt` function performs a linear transformation on `data` over Mod-256 field.

So we can write `decrypt` function as:

```python
def decrypt(data):
    result = ''
    for d in data:
        d = ord(d)
        d = (d - 0x1b) % 256
        d = (d * 81) % 256      # 81 = 0xb1 ^ -1 (mod 256)
        result += chr(d)
    return result
```
